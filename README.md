# @tuya-rn/szos-hooks-sdk

## 介绍

一个兴趣使然的项目

## 开始

安装依赖

```bash
$ npm i
# or
yarn
```

启动服务,

```bash
$ npm start
# or
$ yarn start
```

文档打包

```bash
$ npm run build
# or
$ yarn build
```

开启测试用例,

```bash
$ npm test
# or
$ yarn test
```

结合 github action、docker、docker-compose 自动部署,

```bash
$ npm run npm:build
# or
$ yarn npm:build
```
